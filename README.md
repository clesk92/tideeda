# TideEDA

TideEDA is a Rust-based tool for reading and processing UHDM (Universal Hardware Design Metamodel) files generated by Surelog.

## Features

- **UHDM File Parsing**: Read and parse UHDM files generated by Surelog
- **High-Level Intermediate Representation (HIR)**: Build a structured representation of hardware designs
- **Bit Blast**: Convert HIR to And-Inverter Graph (AIG)
- **Export**: Export AIG to various formats
- **Python Bindings**: Python interface for easy integration

## Project Structure

```
TideEDA/
├── crates/
│   ├── uhdm-rs/      # Core UHDM parsing library
│   ├── hir/           # High-level intermediate representation
│   ├── bitblast/       # Bit blast to AIG
│   ├── export/        # Export functionality
│   ├── pybindings/     # Python bindings
│   └── cli/           # Command-line interface
├── tests/             # Test files
└── Cargo.toml         # Workspace configuration
```

## Installation

### Prerequisites

- Rust (latest stable version)
- Surelog (for generating UHDM files)

### Building

```bash
cargo build --release
```

## Usage

### Command Line

```bash
cargo run --bin tideeda <uhdm_file>
```

Example:
```bash
cargo run --bin tideeda slpp_all/surelog.uhdm
```

### Python API

```python
import tideeda

tideeda.synthesize("input.uhdm", "output.aig")
```

## Development

### Running Tests

```bash
cargo test
```

### Building Individual Crates

```bash
cargo build -p uhdm-rs
cargo build -p hir
cargo build -p bitblast
cargo build -p export
cargo build -p pybindings
cargo build -p tideeda-cli
```

# TideEDA

TideEDA 是一个基于 Rust 的工具，用于读取和处理由 Surelog 生成的 UHDM（通用硬件设计元模型）文件。

## 功能特性

- **UHDM 文件解析**：读取和解析由 Surelog 生成的 UHDM 文件
- **高级中间表示（HIR）**：构建硬件设计的结构化表示
- **位爆破**：将 HIR 转换为与非图（AIG）
- **导出**：将 AIG 导出为各种格式
- **Python 绑定**：提供 Python 接口，便于集成

## 项目结构

TideEDA/
